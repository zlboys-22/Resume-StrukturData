INSERTION SORT¶

Pengertian Insertion Sort¶

Insertion Sort adalah salah satu algoritma sorting (pengurutan data) yang bekerja dengan cara menyisipkan data ke posisi yang benar secara bertahap. Algoritma ini mengurutkan data sedikit demi sedikit hingga seluruh data menjadi terurut.

Konsep Insertion Sort mirip seperti saat seseorang menyusun kartu remi di tangan. Kartu akan diambil satu per satu lalu dimasukkan ke posisi yang sesuai sehingga kartu tetap dalam keadaan terurut.

Insertion Sort termasuk algoritma sorting yang:

  • sederhana,
  • mudah dipahami,
  • dan sangat cocok dipelajari oleh pemula.

Walaupun sederhana, algoritma ini cukup efisien untuk data berukuran kecil atau data yang hampir terurut.

Cara Kerja Insertion Sort¶

Insertion Sort bekerja dengan membagi data menjadi dua bagian:

  1. bagian yang sudah terurut,
  2. dan bagian yang belum terurut.

Pada awal proses:

  • data pertama dianggap sudah terurut,
  • kemudian data berikutnya diambil dan dibandingkan dengan data sebelumnya,
  • lalu disisipkan pada posisi yang benar.

Proses ini dilakukan terus sampai semua data selesai diurutkan.

Ilustrasi Cara Kerja Insertion Sort¶

Contoh data:

[7, 4, 5, 2]

Langkah 1¶

Data pertama dianggap sudah terurut.

[7] [4, 5, 2]

Angka 4 dibandingkan dengan 7.

Karena 4 lebih kecil dari 7, maka 4 dipindahkan ke depan.

Hasil:

[4, 7] [5, 2]

Langkah 2¶

Ambil angka 5.

Bandingkan dengan 7:

  • 5 lebih kecil → geser 7.

Bandingkan dengan 4:

  • 5 lebih besar → berhenti.

Masukkan 5 ke posisi yang benar.

Hasil:

[4, 5, 7] [2]

Langkah 3¶

Ambil angka 2.

Bandingkan dengan:

  • 7 → geser,
  • 5 → geser,
  • 4 → geser.

Masukkan 2 di depan.

Hasil akhir:

[2, 4, 5, 7]

Fungsi dan Tujuan Insertion Sort¶

Insertion Sort digunakan untuk:

  • mengurutkan data angka,
  • mengurutkan huruf,
  • mengurutkan nama,
  • dan berbagai jenis data lainnya.

Tujuan penggunaan Insertion Sort:

  1. membuat data lebih rapi,
  2. mempercepat pencarian data,
  3. membantu pengolahan data,
  4. dan mempermudah analisis data.

Karakteristik Insertion Sort¶

1. Stable Sort¶

Insertion Sort termasuk algoritma stable sorting.

Artinya: data yang memiliki nilai sama tidak akan berubah urutannya.

2. In-Place Sorting¶

Insertion Sort tidak membutuhkan banyak memori tambahan karena proses sorting dilakukan langsung pada data asli.

3. Adaptif¶

Insertion Sort bekerja lebih cepat jika data sudah hampir terurut.

Kelebihan Insertion Sort¶

1. Mudah Dipahami¶

Algoritma ini memiliki langkah yang sederhana sehingga cocok untuk pemula.

2. Cocok Untuk Data Kecil¶

Insertion Sort cukup cepat jika jumlah data sedikit.

3. Efisien Untuk Data Hampir Terurut¶

Jika data sudah hampir urut, proses sorting menjadi lebih cepat.

Contoh:

[1, 2, 3, 5, 4]

Hanya membutuhkan sedikit pergeseran.

4. Tidak Membutuhkan Banyak Memori¶

Karena sorting dilakukan langsung pada array yang sama.

Kekurangan Insertion Sort¶

1. Lambat Untuk Data Besar¶

Jika jumlah data sangat banyak, proses menjadi lambat karena banyak pergeseran data.

2. Banyak Perbandingan dan Pergeseran¶

Pada kondisi terburuk, data harus digeser berkali-kali.

3. Kurang Efisien Dibanding Algoritma Modern¶

Untuk data besar, algoritma seperti:

  • Merge Sort,
  • Quick Sort,
  • Heap Sort

lebih cepat dibanding Insertion Sort.

Kompleksitas Waktu Insertion Sort¶

Best Case¶

Terjadi jika data sudah terurut.

Kompleksitas:

O(n)

Karena hanya perlu satu kali pengecekan.

Worst Case¶

Terjadi jika data terurut terbalik.

Contoh:

[9, 8, 7, 6, 5]

Kompleksitas:

O(n^2)

Karena setiap data harus digeser berkali-kali.

Average Case¶

Kompleksitas rata-rata:

O(n^2)

Kompleksitas Memori¶

Insertion Sort memiliki kompleksitas memori:

O(1)

Artinya hanya membutuhkan sedikit memori tambahan.

Algoritma Insertion Sort¶

Mulai dari data ke-2

Simpan data saat ini

Bandingkan dengan data sebelumnya

Jika lebih kecil:
    geser data sebelumnya

Ulangi sampai posisi benar ditemukan

Masukkan data ke posisi yang benar

Contoh Program Insertion Sort Python¶

data = [7, 4, 5, 2]

for i in range(1, len(data)):

    key = data[i]
    j = i - 1

    while j >= 0 and key < data[j]:

        data[j + 1] = data[j]
        j -= 1

    data[j + 1] = key

print("Hasil sorting:", data)

Penjelasan Program¶

1. Data Awal¶

data = [7, 4, 5, 2]

Data yang akan diurutkan.

2. Perulangan¶

for i in range(1, len(data)):

Perulangan dimulai dari index ke-1 karena data pertama dianggap sudah urut.

3. Menyimpan Data¶

key = data[i]

Data yang akan disisipkan disimpan dalam variabel key.

4. Membandingkan Data¶

while j >= 0 and key < data[j]:

Membandingkan data dengan elemen sebelumnya.

5. Menggeser Data¶

data[j + 1] = data[j]

Jika data sebelumnya lebih besar, maka digeser ke kanan.

6. Menyisipkan Data¶

data[j + 1] = key

Data dimasukkan ke posisi yang benar.

Output Program¶

Hasil sorting: [2, 4, 5, 7]

Contoh Penggunaan Insertion Sort Dalam Kehidupan Nyata¶

1. Mengurutkan Kartu¶

Saat bermain kartu, biasanya kartu disusun dari kecil ke besar secara bertahap.

2. Mengurutkan Nilai¶

Data nilai siswa dapat diurutkan dari terkecil ke terbesar.

3. Mengurutkan Nama¶

Daftar nama dapat diurutkan berdasarkan abjad.

Kesimpulan¶

Insertion Sort adalah algoritma sorting sederhana yang bekerja dengan cara menyisipkan data ke posisi yang benar secara bertahap.

Algoritma ini:

  • mudah dipahami,
  • cocok untuk pemula,
  • efisien pada data kecil,
  • dan baik digunakan pada data yang hampir terurut.

Namun, untuk data besar, Insertion Sort kurang efisien karena membutuhkan banyak proses perbandingan dan pergeseran data.

def urutkan_kartu(arr):
    for i in range(1, len(arr)):
        kartu_aktif = arr[i]
        j = i - 1

        while j >= 0 and kartu_aktif < arr[j]:
            arr[j + 1] = arr[j]
            j = j - 1            
        arr[j + 1] = kartu_aktif
    return arr
