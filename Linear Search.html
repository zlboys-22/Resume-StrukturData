LINEAR SEARCH¶

Pengertian Linear Search¶

Linear Search adalah algoritma searching (pencarian data) yang bekerja dengan cara memeriksa data satu per satu secara berurutan mulai dari data pertama hingga data yang dicari ditemukan atau sampai seluruh data selesai diperiksa.

Linear Search merupakan metode pencarian paling sederhana dalam Struktur Data dan Algoritma sehingga sangat mudah dipahami oleh pemula.

Algoritma ini disebut “linear” karena proses pencarian dilakukan secara lurus atau berurutan dari awal sampai akhir data.

Konsep Dasar Linear Search¶

Pada Linear Search:

  • komputer akan membandingkan data yang dicari dengan setiap elemen data,
  • dimulai dari index pertama,
  • lalu bergerak satu per satu hingga data ditemukan.

Jika data ditemukan:

  • proses pencarian berhenti.

Jika data tidak ditemukan:

  • sistem akan memberikan informasi bahwa data tidak ada.

Cara Kerja Linear Search¶

Misalnya terdapat data:

[15, 20, 35, 40, 50]

Data yang dicari:

35

Langkah Pencarian¶

Langkah 1¶

Bandingkan:

  • 35 dengan 15

Karena tidak sama → lanjut.

Langkah 2¶

Bandingkan:

  • 35 dengan 20

Karena tidak sama → lanjut.

Langkah 3¶

Bandingkan:

  • 35 dengan 35

Karena sama → data ditemukan.

Ilustrasi Linear Search¶

Index : 0   1   2   3   4
Data  : 15  20  35  40  50

Pencarian:

  • cek index 0
  • cek index 1
  • cek index 2 → ditemukan

Karakteristik Linear Search¶

1. Pencarian Berurutan¶

Data dicek satu per satu dari awal hingga akhir.

2. Tidak Harus Data Terurut¶

Linear Search dapat digunakan pada:

  • data acak,
  • maupun data yang sudah terurut.

3. Sederhana¶

Algoritma ini sangat mudah dibuat dan dipahami.

4. Cocok Untuk Data Kecil¶

Linear Search cukup efektif jika jumlah data sedikit.

Fungsi Linear Search¶

Linear Search digunakan untuk:

  • mencari angka,
  • mencari nama,
  • mencari data mahasiswa,
  • mencari produk,
  • mencari file,
  • dan data lainnya.

Tujuan Linear Search¶

Tujuan utama Linear Search adalah:

  • menemukan data tertentu,
  • menentukan posisi data,
  • dan memastikan apakah data tersedia atau tidak.

Kelebihan Linear Search¶

1. Mudah Dipahami¶

Linear Search sangat cocok dipelajari pemula karena langkahnya sederhana.

2. Mudah Diimplementasikan¶

Kode program Linear Search cukup pendek dan sederhana.

3. Tidak Memerlukan Data Terurut¶

Berbeda dengan beberapa algoritma lain, Linear Search tetap bisa digunakan meskipun data acak.

Contoh:

[40, 10, 90, 25]

Tetap dapat dicari menggunakan Linear Search.

4. Cocok Untuk Data Sedikit¶

Jika jumlah data kecil, Linear Search bekerja cukup baik.

Kekurangan Linear Search¶

1. Lambat Untuk Data Besar¶

Jika jumlah data sangat banyak, pencarian menjadi lebih lama karena harus memeriksa satu per satu.

2. Kurang Efisien¶

Linear Search membutuhkan banyak perbandingan data.

3. Waktu Pencarian Tidak Stabil¶

Kadang data ditemukan cepat, kadang harus memeriksa seluruh data.

Kompleksitas Waktu Linear Search¶

Best Case¶

Terjadi ketika data ditemukan di awal.

Contoh:

[10, 20, 30]

Mencari:

  • 10

Hanya membutuhkan satu kali pengecekan.

Kompleksitas:

O(1)

Worst Case¶

Terjadi ketika:

  • data berada di akhir,
  • atau data tidak ditemukan.

Contoh:

[10, 20, 30, 40, 50]

Mencari:

  • 50
  • atau 100

Harus memeriksa semua data.

Kompleksitas:

O(n)

Average Case¶

Kompleksitas rata-rata:

O(n)

Kompleksitas Memori¶

Linear Search memiliki kompleksitas memori:

O(1)

Karena tidak membutuhkan memori tambahan yang besar.

Pseudocode Linear Search¶

Mulai dari data pertama

Bandingkan data dengan nilai yang dicari

Jika sama:
    tampilkan data ditemukan

Jika tidak:
    lanjut ke data berikutnya

Ulangi sampai data habis

Contoh Program Linear Search Python¶

data = [15, 20, 35, 40, 50]

cari = 35

ditemukan = False

for i in range(len(data)):

    if data[i] == cari:
        ditemukan = True
        posisi = i
        break

if ditemukan:
    print("Data ditemukan di index", posisi)
else:
    print("Data tidak ditemukan")

Penjelasan Program¶

1. Data¶

data = [15, 20, 35, 40, 50]

Data yang akan dicari.

2. Data Yang Dicari¶

cari = 35

Nilai yang ingin ditemukan.

3. Perulangan¶

for i in range(len(data)):

Digunakan untuk memeriksa seluruh data satu per satu.

4. Pengecekan¶

if data[i] == cari:

Memeriksa apakah data sama dengan nilai yang dicari.

5. Menghentikan Pencarian¶

break

Digunakan agar program berhenti saat data ditemukan.

Output Program¶

Data ditemukan di index 2

Contoh Linear Search Dalam Kehidupan Nyata¶

1. Mencari Nama Di Daftar Hadir¶

Guru memeriksa nama siswa satu per satu sampai nama ditemukan.

2. Mencari Buku¶

Mencari buku di rak tanpa urutan tertentu.

3. Mencari Kontak¶

Mencari nama kontak secara manual dari atas ke bawah.

4. Mencari Barang¶

Mencari barang di gudang satu per satu.

Kapan Linear Search Digunakan?¶

Linear Search cocok digunakan ketika:

  • jumlah data sedikit,
  • data belum terurut,
  • proses pencarian sederhana,
  • dan implementasi mudah lebih diutamakan.

Perbedaan Linear Search Dengan Searching Lain¶

Linear Search Searching Lain
Sederhana Lebih kompleks
Mudah dipahami Kadang sulit dipahami
Tidak perlu data terurut Beberapa harus terurut
Lambat pada data besar Lebih cepat pada data besar

Kesimpulan¶

Linear Search adalah algoritma pencarian sederhana yang bekerja dengan cara memeriksa data satu per satu secara berurutan hingga data ditemukan.

Algoritma ini:

  • mudah dipahami,
  • mudah dibuat,
  • cocok untuk pemula,
  • dan dapat digunakan pada data yang belum terurut.

Namun, Linear Search kurang efisien untuk data berukuran besar karena membutuhkan banyak proses perbandingan data.

def linear_search(data, cari):
    for i in range(len(data)):
        if data[i] == cari:
            return i
    return -1

angka = [10, 20, 30, 40, 50]
target = 30

hasil = linear_search(angka, target)

if hasil != -1:
    print("Data ditemukan di index", hasil)
else:
    print("Data tidak ditemukan")
