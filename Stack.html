Stack¶

Stack adalah struktur data linier yang bekerja dengan prinsip LIFO (Last In, First Out), yaitu:¶

  • Data yang terakhir dimasukkan akan menjadi data yang pertama dikeluarkan.
  • Akses data hanya dilakukan pada satu sisi, yaitu bagian atas (top) stack.

Analogi Stack:

Tumpukan piring → piring terakhir yang ditaruh akan diambil lebih dulu

Tumpukan buku → buku paling atas adalah yang pertama diambil

Ciri utama stack antara lain:¶

  • Akses data bersifat terbatas
  • Elemen hanya dapat ditambah atau dihapus dari satu ujung
  • Tidak mendukung akses acak (random access)
  • Sangat cocok untuk proses rekursif dan backtracking

Operasi Dasar pada Stack¶

  • Push Menambahkan elemen ke bagian atas stack.
  • Pop Menghapus elemen paling atas dari stack.
  • Peek / Top Melihat elemen teratas tanpa menghapusnya.
  • isEmpty Mengecek apakah stack kosong.
  • Size Mengetahui jumlah elemen dalam stack.

Parkiran Mobil (Model LIFO)¶

Sebuah parkiran hanya memiliki satu jalur masuk dan keluar (model stack).

Ketentuan:

Mobil masuk → push

Mobil keluar → pop

Jika parkiran penuh (maks 5 mobil) → tampilkan pesan

Buat program simulasi parkiran tersebut.

In [7]:
parkiran = []
kapasitas_maks = 5

while True:
    print("\n=== MENU PARKIRAN ===")
    print("1. Mobil Masuk")
    print("2. Mobil Keluar")
    print("3. Lihat Parkiran")
    print("4. Keluar Program")

    pilihan = input("Pilih menu (1-4): ")

    if pilihan == "1":
        if len(parkiran) >= kapasitas_maks:
            print("Parkiran penuh! Tidak bisa menambah mobil.")
        else:
            mobil = input("Masukkan nomor mobil: ")
            parkiran.append(mobil)
            print(f"Mobil {mobil} masuk ke parkiran.")

    elif pilihan == "2":
        if len(parkiran) == 0:
            print("Parkiran kosong! Tidak ada mobil yang keluar.")
        else:
            mobil_keluar = parkiran.pop()
            print(f"Mobil {mobil_keluar} keluar dari parkiran.")

    elif pilihan == "3":
        if len(parkiran) == 0:
            print("Parkiran kosong.")
        else:
            print("Isi parkiran (urutan masuk):")
            i = 1
            for mobil in parkiran:
                print(f"{i}. {mobil}")
                i += 1

    elif pilihan == "4":
        print("Program selesai. Terima kasih!")
        break

    else:
        print("Pilihan tidak valid!")

stack_buku = []

while True:
    print("\n=== MENU PERPUSTAKAAN ===")
    print("1. Tambah Buku")
    print("2. Ambil Buku Paling Atas")
    print("3. Tampilkan Buku Paling Atas")
    print("4. Tampilkan Seluruh Buku")
    print("5. Keluar")

    pilihan = input("Pilih menu (1-5): ")

    if pilihan == "1":
        buku = input("Masukkan judul buku: ")
        stack_buku.append(buku)
        print(f"Buku '{buku}' berhasil ditambahkan.")

    elif pilihan == "2":
        if len(stack_buku) == 0:
            print("Stack kosong, tidak ada buku yang bisa diambil.")
        else:
            buku = stack_buku.pop()
            print(f"Buku '{buku}' diambil dari tumpukan.")

    elif pilihan == "3":
        if len(stack_buku) == 0:
            print("Stack kosong.")
        else:
            print("Buku paling atas:", stack_buku[-1])

    elif pilihan == "4":
        if len(stack_buku) == 0:
            print("Tidak ada buku dalam perpustakaan.")
        else:
            print("Daftar buku (dari atas ke bawah):")
            for buku in reversed(stack_buku):
                print("-", buku)

    elif pilihan == "5":
        print("Program selesai. Terima kasih!")
        break

    else:
        print("Pilihan tidak valid!")
