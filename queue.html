queue¶

Queue (Antrian) adalah salah satu struktur data dalam ilmu komputer yang digunakan untuk menyimpan sekumpulan data dengan prinsip FIFO (First In First Out), artinya data yang pertama masuk akan menjadi data pertama yang keluar.¶

Queue adalah struktur data linear di mana penambahan elemen dilakukan di bagian belakang (rear), sedangkan penghapusan elemen dilakukan di bagian depan (front).¶

Secara sederhana, queue bisa dianalogikan seperti antrian di kasir atau bank:¶

- Orang yang datang lebih dulu → dilayani lebih dulu¶

- Orang yang datang belakangan → menunggu giliran¶

Karakteristik Queue¶

FIFO (First In First Out)

  • Elemen pertama yang masuk akan keluar pertama.

Dua ujung utama:

  • Front (depan) → tempat menghapus data
  • Rear (belakang) → tempat menambah data

Operasi terbatas

  • Tidak seperti array, kita tidak bisa sembarang mengakses elemen di tengah.

Operasi Dasar Queue¶

Enqueue (menambah data)¶

  • Menambahkan elemen ke bagian belakang (rear).

Dequeue (menghapus data)¶

  • Menghapus elemen dari bagian depan (front).

Peek / Front¶

  • Melihat elemen paling depan tanpa menghapusnya.

IsEmpty¶

  • Mengecek apakah queue kosong.

IsFull (untuk queue statis)¶

  • Mengecek apakah queue sudah penuh.

queue = []

def enqueue(data):
    queue.append(data)
    print(data, "ditambahkan ke queue")

def dequeue():
    if len(queue) == 0:
        print("Queue kosong!")
    else:
        data = queue.pop(0)
        print(data, "dihapus dari queue")

def tampilkan():
    print("Isi Queue:", queue)

class Queue:
   def __init__(self):
       self.items = []
 
   def is_empty(self):
       return len(self.items) == 0
 
   def enqueue(self, item):
       self.items.append(item)
 
   def dequeue(self):
       if not self.is_empty():
           return self.items.pop(0)
       else:
           return "Queue is empty"
 
   def size(self):
       return len(self.items)
 
antrian = Queue()
 
antrian.enqueue("Pelanggan 1")
antrian.enqueue("Pelanggan 2")
antrian.enqueue("Pelanggan 3")
 
print("Ukuran antrian:", antrian.size())
 
print("Panggilan ke kasir:", antrian.dequeue())
 
print("Ukuran antrian setelah panggilan pertama:", antrian.size())

Ukuran antrian: 3
Panggilan ke kasir: Pelanggan 1
Ukuran antrian setelah panggilan pertama: 2
